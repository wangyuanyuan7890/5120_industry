## Instructions

To install all packages

```bash
npm install
```

### Run in development mode

```bash
npm run dev
```

### Run in production mode

First build the production ready files

```bash
npm run build
```

Then run your production server

```bash
npm run start
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
